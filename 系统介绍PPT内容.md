# 幼儿园管理系统 - 系统介绍

## 1. 系统概述

### 1.1 系统简介
- 幼儿园管理系统是一个面向幼儿园日常运营管理的综合信息化平台
- 提供幼儿信息管理、班级管理、教师管理、选区管理等核心功能
- 支持多级权限管理（系统管理员、园长、教师）

### 1.2 系统目标
- 提升幼儿园管理效率，实现信息化管理
- 规范幼儿、班级、教师等核心数据管理
- 提供数据可视化，辅助决策分析
- 简化日常操作流程，降低管理成本

## 2. 功能模块介绍

### 3.1 用户认证与权限管理
- **登录与注销**: 支持验证码登录，确保系统安全
- **角色权限**: 系统管理员、园长、教师三种角色
- **动态路由**: 根据用户角色动态生成可访问菜单
- **Token管理**: 自动处理Token过期和刷新

### 3.2 仪表盘（Dashboard）
- **系统概览**: 显示核心业务指标
- **统计数据**: 幼儿总数、班级数量、教师总数、今日出勤率等
- **趋势图表**: 幼儿人数趋势、班级人数分布、选区幼儿分布
- **实时数据**: 最近活动动态展示
- **数据刷新**: 支持手动刷新获取最新数据

### 3.3 幼儿管理
- **基础功能**: 新增、编辑、删除、查询幼儿信息
- **搜索筛选**: 支持按姓名、班级、家长姓名、联系电话搜索
- **详细信息**: 幼儿基本信息、家长信息、紧急联系人等
- **头像管理**: 支持上传、预览、删除幼儿头像
- **状态管理**: 支持标记在读/已毕业状态
- **分页显示**: 支持大数据量分页展示

### 3.4 班级管理
- **基础功能**: 新增、编辑、删除、查询班级
- **班级类型**: 支持小班、中班、大班分类管理
- **容量设置**: 支持设置班级最大容量
- **位置管理**: 记录教室位置信息
- **数据统计**: 显示班级幼儿人数统计
- **关联管理**: 与幼儿、教师模块关联

### 3.5 教师管理
- **基础功能**: 新增、编辑、删除、查询教师信息
- **职位管理**: 支持班主任、配班老师、生活老师、园长等职位
- **个人信息**: 工号、姓名、性别、联系方式等详细信息
- **班级分配**: 支持将教师分配到不同班级
- **入职管理**: 记录教师入职日期等信息

### 3.6 选区管理
- **区域划分**: 支持幼儿园选区管理
- **幼儿分配**: 管理选区与幼儿的对应关系
- **数据统计**: 选区幼儿数量统计

## 4. 页面功能详解

### 4.1 登录页面
- **界面组成**: 系统Logo、登录表单、验证码图片
- **功能说明**: 用户身份验证入口，支持验证码防止暴力破解
- **技术亮点**: 验证码自动刷新、表单验证、错误提示

### 4.2 仪表盘页面
- **界面组成**: 欢迎区、统计卡片、图表区、最近活动表格
- **功能说明**: 系统核心数据概览，提供关键指标快速查看
- **统计指标**:
  - 幼儿总数及变化率
  - 班级数量
  - 教师总数
  - 今日出勤率
  - 选区总数
  - 已分配/未分配选区幼儿数
- **图表类型**: 折线图、柱状图、饼图等多种可视化方式

### 4.3 幼儿管理页面
- **界面组成**: 搜索栏、数据表格、分页控件、新增/编辑对话框
- **搜索功能**: 多条件组合查询
- **表格展示**: ID、姓名、性别、出生日期、年龄、班级、家长信息、状态等
- **操作功能**:
  - 新增: 填写完整幼儿及家长信息，上传头像
  - 编辑: 修改现有幼儿信息
  - 删除: 移除幼儿信息（带确认提示）
  - 查看: 查看幼儿详细信息
- **表单验证**: 完善的字段验证机制
- **头像处理**: 支持图片预览、上传限制（格式、大小）

### 4.4 班级管理页面
- **界面组成**: 搜索栏、数据表格、分页控件、新增/编辑对话框
- **搜索功能**: 按班级名称、班级类型搜索
- **表格展示**: ID、班级名称、幼儿园、班级类型、幼儿人数、最大容量、创建时间等
- **操作功能**:
  - 新增: 创建新班级，设置类型、容量等信息
  - 编辑: 修改班级信息
  - 删除: 移除班级（带确认提示）
- **类型管理**: 小班、中班、大班三种类型，不同类型有不同样式标识

### 4.5 教师管理页面
- **界面组成**: 搜索栏、数据表格、分页控件、新增/编辑对话框
- **搜索功能**: 按姓名、工号、手机号、职位搜索
- **表格展示**: ID、工号、姓名、性别、手机号、邮箱、职位、所在班级、入职日期等
- **操作功能**:
  - 新增: 添加教师信息，分配职位和班级
  - 编辑: 修改教师信息
  - 移除: 移除教师（带确认提示）
- **职位管理**: 支持班主任、配班老师、生活老师、园长等不同职位

## 5. 业务流程

### 5.1 幼儿信息管理流程
1. **新增流程**: 填写幼儿基本信息 → 填写家长信息 → 上传头像 → 分配班级 → 提交保存
2. **编辑流程**: 选择幼儿 → 修改信息 → 保存更新
3. **查询流程**: 输入搜索条件 → 点击查询 → 查看结果列表 → 分页浏览

### 5.2 班级管理流程
1. **创建班级**: 填写班级名称 → 选择班级类型 → 设置容量 → 保存创建
2. **班级配置**: 分配教师 → 录入幼儿 → 管理班级资源

### 5.3 教师管理流程
1. **教师入职**: 录入基本信息 → 分配职位 → 分配班级（如适用）→ 保存记录
2. **职位调整**: 编辑教师信息 → 修改职位 → 更新班级分配

### 5.4 用户权限流程
1. **登录认证**: 输入账号密码 → 验证验证码 → 验证身份 → 获取Token → 进入系统
2. **权限控制**: 根据用户角色 → 动态生成菜单 → 控制功能访问权限
